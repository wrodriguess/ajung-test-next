# Como instalar e executar o projeto:

## 1 - Clone o repositório em sua máquina
### `git clone https://github.com/wrodriguess/ajung-test-next.git`

## 2 - Acesse a página do projeto
### `cd ajung-test-next`

## 3 - Instale as dependenicas
### `npm install`

## 4 - Execute o projeto
### `npm run dev`
